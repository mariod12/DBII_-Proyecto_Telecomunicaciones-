# DBII_Proyecto_Telecomunicaciones-
Proyecto de BD2 sobre una base de datos para telecomunicaciones, con procesos ETL y visualización de datos en Power BI.

## Descripción

Proyecto académico desarrollado para la asignatura **Base de Datos II**, enfocado en el diseño e implementación de una base de datos para el sector de **telecomunicaciones**.

## Objetivo

Diseñar una solución de almacenamiento y análisis de datos que permita organizar, transformar y visualizar información relevante para la toma de decisiones.

## Tecnologías utilizadas

* **SQL** – Creación del modelo de datos y procesos ETL
* **Power BI** – Visualización y análisis de datos

## Modelo de datos

* **Modelo estrella (Star Schema)**
* Separación entre tablas de hechos y dimensiones para optimizar consultas analíticas.

## Procesos ETL

* Extracción, transformación y carga de datos implementadas completamente en **SQL**.
* Limpieza, normalización y preparación de los datos para análisis.

## Visualización

* Creación de **dashboards interactivos en Power BI** para la presentación de indicadores clave del sector de telecomunicaciones.
